# LifePlus Catálogo Premium - 118 Productos de Bienestar

🌿 **Catálogo web moderno y completo de productos LifePlus**

## 📋 Descripción

Este proyecto contiene un catálogo web profesional para los 118 productos premium de bienestar de LifePlus. Diseñado con una interfaz moderna, responsiva y siguiendo los colores corporativos de la marca.

## ✨ Características

- **118 Productos Premium**: Todos los productos del catálogo completo
- **9 Categorías Especializadas**: Nutricionales, Proteínas, Nutrición Deportiva, etc.
- **Diseño Moderno**: Interfaz limpia y profesional con colores LifePlus
- **Totalmente Responsivo**: Funciona perfectamente en todos los dispositivos
- **Enlaces Directos**: Cada producto tiene un botón directo a la tienda
- **Navegación Suave**: Menú de navegación con scroll suave entre categorías

## 🎨 Colores y Estilo

El diseño sigue la imagen corporativa de LifePlus con:
- Paleta de colores verde corporativo
- Tipografía Montserrat profesional
- Gradientes sutiles y efectos modernos
- Tarjetas de productos interactivas

## 🚀 Cómo Usar

1. **Abrir el Catálogo**:
   - Haz doble clic en `catalogo_lifeplus.html`
   - O abrelo en tu navegador web preferido

2. **Navegación**:
   - Usa el menú de navegación para saltar entre categorías
   - Haz scroll para ver todos los productos
   - Haz clic en "Comprar en Tienda" para cada producto

3. **Enlaces**:
   - Todos los productos enlazan a: https://ww1.lifeplus.com/SHVCB5/S/
